OMNICheck Database Installation Wizard

Step 1: Start PostgreSQL Installer

Double-click the PostgreSQL installer file. Click Next to continue.

Step 2: Installation Directory

Choose where PostgreSQL will be installed (default recommended). Click Next.

Step 3: Select Components

Keep defaults (Server, pgAdmin, Command Line Tools). Click Next.

Step 4: Data Directory

Choose where to store database data. Default is recommended. Click Next.

Step 5: Superuser Password

Enter a password for the PostgreSQL superuser postgres. Remember this for later use.

Step 6: Port Selection

Leave default port as 5432. Click Next.

Step 7: Locale

Select locale (default recommended). Click Next.

Step 8: Pre-Installation Summary

Review your settings. If correct, click Next.

Step 9: Installing

The installer copies files and configures PostgreSQL. Please wait.

Step 10: Installation Complete

Click Finish to close the installer.

Step 11: Install pgAdmin

Run the pgAdmin 4 installer. Accept default options unless required otherwise.

Step 12: Install ODBC Driver

Run the PostgreSQL Unicode (x64) ODBC driver installer. Verify it appears in ODBC Data Sources.

Step 13: Configure OMNICheck Server in pgAdmin

Open pgAdmin, add a new server with these details:

  • Server name: OMNICheck Server
  • Host: localhost
  • Port: 5432
  • User: ator
  • Password: Pass

Step 14: Verify Installation

In pgAdmin Query Tool, run:

SELECT version();
